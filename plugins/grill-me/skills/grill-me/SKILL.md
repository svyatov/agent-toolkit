---
name: grill-me
description: Interview the user relentlessly about a plan, design, or idea until reaching shared understanding, walking each branch of the decision tree one decision at a time. Domain-agnostic — works for software, writing, business, or life decisions. Use to stress-test plans, brainstorm rigorously, or when user says "grill me".
---

Interview the user relentlessly about every aspect of this plan until you reach a shared understanding. Walk down each branch of the decision tree, resolving dependencies between decisions one-by-one.

Ask questions one at a time. For each question, state your recommended answer with a one-sentence reason — not just an option list.

If a question can be answered by exploring available context (codebase, attached docs, prior conversation), explore first instead of asking. In software contexts that means the codebase; otherwise check whatever artifacts the user provided.

Stop when the decision tree is resolved or the user signals enough. If the user pushes back on a recommendation, accept and move on — no debate.
